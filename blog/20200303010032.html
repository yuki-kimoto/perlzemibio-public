塩基配列の長さを取得

塩基配列の長さを取得するにはlength文を使用します。 

length 文字列互換型;

塩基配列から、文字列の長さを取り出すサンプルです。 

my $sequence = "ATGCAGCCC";
my $length = length $sequence;

塩基配列の長さを取得するPerlから呼び出すサンプル

# CountSequence.spvm
package CountSequence {
  sub count : int ($sequence : string) {
    return length $sequence;
  }
}

use strict;
use warnings;
use SPVM "CountSequence";

my $sequence = "ATGCAGCCC";
print CountSequence->count($sequence), "\n";
# Output
# 9
